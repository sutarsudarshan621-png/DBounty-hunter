# DBounty Hunters

Decentralized bounty marketplace on Stellar/Soroban. Monorepo containing:

- `frontend/` — React + TailwindCSS + Three.js + GSAP client
- `backend/` — Rust (Axum) API gateway, PostgreSQL + Redis
- `contracts/` — Soroban smart contracts (escrow, rewards, badges)
- `infra/` — Docker/deployment configuration
- `docs/` — Architecture, API, and contract specs

See `docs/ARCHITECTURE.md` for the full system design.
